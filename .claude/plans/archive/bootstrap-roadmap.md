# Bootstrap Roadmap

## Goal

Turn this repo into the standalone XSD parsing satellite for the XML ecosystem.

## Phase 1

- bootstrap package manifests and AI workflow files
- extract standalone XSD model
- extract standalone XSD parser
- add local include/import resolver
- add focused parser tests

## Exit Criteria

- `SwiftXMLSchema` builds as a standalone package
- public API is SOAP-free
- recursive local include/import parsing works
